Number Theory

จำนวน

จำนวนจริง

  • R
  • Real number
  • ทุกจำนวน ยกเว้น รูทติดลบ และ เลขที่มีส่วนเป็นศูนย์

จำนวนเต็ม

  • Z
  • Integer number
  • ..., -2, -1, 0, 1, 2, ...

จำนวนนับ

  • N
  • Natural or counting number
  • จำนวนเต็มบวก
  • 1, 2, 3, ...

จำนวนตรรกยะ

  • Q
  • Rational number
  • เขียนในรูปเศษส่วนได้, ทศนิยมรู้จบ, ทศนิยมไม่รู้จบแต่ซ้ำ
  • 2 ⁄ 3 ,- 1 ⁄ 4 , 0.1, 3.3333...

จำนวนอตรรกยะ

  • Q'
  • Irrational number
  • เขียนในรูปเศษส่วนไม่ได้
  • √2, √3, π

เอกลักษณ์การบวก

  • บวกแล้วได้จำนวนเดิมคือ 0
  • a + 0 = 0

เอกลักษณ์การคูณ

  • คูณแล้วได้จำนวนเดิมคือ 1
  • a x 1 = 1

อินเวอร์สการบวก

  • จำนวนที่บวกแล้วได้เอกลักษณ์คือ -a
  • -a + a = 0

อินเวอร์สการคูณ

  • จำนวนที่คูณแล้วได้เอกลักษณ์คือ 1/a
  • 1/a x a = 1 เมื่อ a ≠ 0

สมบัติปิด

  • นำสมาชิกในเซตใดๆมาดำเนินการกัน แล้วผลลัพธ์ยังเป็นสมาชิกในเซต
  • เซตของจำนวนนับมี สมบัติปิด สำหรับการบวกและการคูณ

หลักการที่ควรรู้

y = |x|

  • x ∈ R, y ≥ 0
  • x เป็นเลขอะไรก็ได้แต่ y ≥ 0

y = a ⁄ x

  • x ≠ 0

y = x 2

  • x ∈ R, y ≥ 0
  • x เป็นเลขอะไรก็ได้แต่ y ≥ 0

y = √x

  • x ≥ 0, y ≥ 0
  • ในรูทห้ามติดลบ ถอดมาก็ห้ามลบ

ค่าสัมบูรณ์ (Absolute)

ทฤษฏีบท

  • |x| ≥ 0
  • |x| = |-x|
  • |xy| = |x| |y|
  • |x/y| = |x|/|y|
  • |x-y| = |y-x|
  • |x+y| ≤ |x|+|y|
  • |x-y| ≥ |x|-|y|
  • |x n | = |x| n เมื่อ n เป็นจำนวนเต็มใดๆ
  • |x| = √x 2

หลักในการแก้สมการ

กำหนดให้ a ≥ 0>

  • ถ้า |x| = a แล้ว x = a หรือ x = -a
  • ถ้า |x| < a แล้ว -a < x < a
  • ถ้า |x| ≤ a แล้ว -a ≤ x ≤ a
  • ถ้า |x| > a แล้ว x < -a หรือ x > a
  • ถ้า |x| ≥ a แล้ว x ≤ -a หรือ x ≥ a
